# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-29

### Added
- Initial release of Sanskrit Search library
- Sandhi-aware text search functionality
- Pratika identification system
- Sandhi splitting rules (vowel, consonant, visarga)
- Multi-document search support
- Search indexing for performance
- Configurable search options
- Browser and Node.js compatibility
- Comprehensive documentation and examples
- Demo page for testing features

### Features
- Direct substring matching
- Sandhi rule handling (compound word search)
- Context extraction around matches
- HTML highlighting support
- Structured data (JSON) search
- Zero external dependencies
